Three takes on youarepayingtoomuch.com

Same calculator math, same compliance copy, same brand system. Three different aesthetic directions for the postcard's landing page.

Brand-respecting redesign. Each version uses the existing brand: Inter (body) + DM Sans (logo wordmark), the brand-green family from tokens.ts (#00A540 et al.), and the real logo asset inlined as SVG. The fee-drag red accent (#B91C1C) matches the existing fee-receipt variant.
  • V1 Fee Receipt — trust through restraint. The calculator is laid out like an itemized printed receipt. Best fit for an audience that reads the WSJ.
  • V2 Vault — trust through depth. Uses the existing fiduciary-upgrade dark variant palette; two-ledger comparison and an animated odometer.
  • V3 Savings Pass — trust through clarity. Boarding-pass meets quiz, with three tap-to-answer questions instead of sliders. Designed thumb-first for QR-code traffic.
