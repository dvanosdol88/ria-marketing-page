# 02 · Structured Columns

**Slug:** `structured-columns`

**One-liner:** Three-column Swiss-style grid: founder voice on left, headline + chart center, QR + receipt-style numbers right.

A disciplined three-column grid drives clear reading order across both sides. A 2pt brand-green keyline marks each column header without flooding any panel with mint. The center column carries the headline and chart; the right column compresses the savings numbers like a small receipt, anchored by the QR. The back uses the same column structure to give the indicia block a clean home above a horizontal divider, with logo + photo + title in the left column, benefit copy center, and QR + domain right.

**Print-safety verdict: Medium-high.** Ink coverage is modest, no gradients, no halftones. The keylines are 2pt — friendly to consumer EDDM presses — but verify the green hits #00A540 cleanly on whatever stock the printer specs.
