# Style B EDDM

Three new EDDM proof concepts built from David's Style B screenshot.

- 01 Hero Gap: postcard headline plus lifetime fee gap and two-path chart.
- 02 Three Taps: simple scan promise plus extra-money poll and explainer tiles.
- 03 Advisor Proof: fee-gap proof paired immediately with David's advisor/fiduciary block.

All proofs use the same 8.75in x 6.5in proof canvas as Style A, with a 0.125in bleed guide. Logo, David photo, and QR code are copied into this round from the Style A finalist asset folder.
