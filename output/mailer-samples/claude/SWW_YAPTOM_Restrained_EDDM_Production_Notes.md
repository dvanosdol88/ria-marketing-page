# SWW YAPTOM Restrained EDDM Mailer

Generated by `scripts/build-eddm-restrained.mjs`.

## Design intent

Restrained / Swiss. Single accent color (deep brand green #00682B). Hairline rules
instead of card backgrounds. No drop shadows. No rounded decoration. Limited type
scale. Rectangular DVO crop. Chart rebuilt from scratch (no card, hairline axes,
inline end-of-line labels).

## Print format

- Trim: 8.5 x 6.25 in landscape, 0.125 in bleed all sides.
- PDF page: 8.75 x 6.5 in. Proof PNGs: 1750 x 1300 px.
- Two pages: 1 = front, 2 = back/mailing side.
